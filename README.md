# Website Berita (HTML + Tailwind CSS)

Project website berita sederhana yang dibuat pakai **HTML** dan **Tailwind CSS (CDN)**.  
Fokus ke tampilan, dan layout.

---

## Fitur

- Berita utama (hero section)
- Sidebar **Trending Sekarang** (sticky)
- Kategori berita (Teknologi, Politik, Bisnis, dll)
- Card berita dengan gambar & kategori
- Search bar, tombol login & subscribe (UI saja)

---

## Tech

- HTML5  
- Tailwind CSS (CDN)

Tanpa JavaScript.

---

## Cara Jalankan

Tinggal buka `index.html`  
atau pakai Live Server di VS Code.

---

## Catatan

Project ini dibuat buat latihan frontend dan ngasah layouting pakai Tailwind.
